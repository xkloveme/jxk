Install

pnpm add jxk

yarn add jxk

npm i jxk

Add plugin to your vite.config.ts or vite.config.js:

// vite.config.ts
import { defineConfig } from 'vite'
import VersionGitPlugin from 'jxk'

export default defineConfig({
  plugins: [
    VersionGitPlugin(),
  ],
})
